\section{Motivation}
\begin{frame}[allowframebreaks]{Motivation}
    \begin{itemize}
        \item \textbf{Why study LRMs?}
        \begin{itemize}
            \item LLMs like GPT-3 and GPT-4 excel at generating text.
            \item However, they often rely on \textbf{pattern matching}, not true reasoning.
            \item Real-world applications like \textbf{math problem-solving}, \textbf{code synthesis}, and \textbf{scientific hypothesis generation} require multi-step reasoning.
        \end{itemize}
        \item \textbf{Examples:}
        \begin{itemize}
            \item \textit{GPT-3 can generate realistic text but fails at solving "If Alice has 3 apples and gives Bob 2, how many does she have?" unless prompted with reasoning steps.}
            \item \textit{DeepMind’s AlphaCode uses reasoning techniques to solve programming problems—pure LLMs fall short.}
        \end{itemize}
    \end{itemize}
\end{frame}
