\section{Summary}
\begin{frame}{Summary}
  \begin{itemize}
    \item[$\blacktriangleright$] Seq2Seq enables mapping input to output sequences of variable lengths
    \item[$\blacktriangleright$] \textbf{Bottleneck:} Fixed-size context vector limits learning capacity
    \item[$\blacktriangleright$] \textbf{Attention:} Improves performance by giving \textbf{adaptive access} to input states
    \item[$\blacktriangleright$] Attention $\rightarrow$ Transformer $\rightarrow$ LLMs
    \item[$\blacktriangleright$] Still evolving: From additive attention to self-attention and beyond
  \end{itemize}
\end{frame}
