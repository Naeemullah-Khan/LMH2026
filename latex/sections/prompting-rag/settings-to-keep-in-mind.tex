\begin{frame}{Settings to Keep in Mind}
  \begin{itemize}
    \item When prompting a language model, you should keep in mind a few settings.
    \vspace{0.5em}
    \item You can get very different results with prompts when using different settings.
    \vspace{0.5em}
    \item One important setting is controlling how \textbf{deterministic} the model is when generating completions for prompts:
      \begin{itemize}
        \item \textbf{Temperature} and \textbf{top\_p} are two important parameters to keep in mind.
        \item Generally, keep these low if you are looking for exact answers.
        \item \dots keep them high if you are looking for more diverse responses.
      \end{itemize}
  \end{itemize}
\end{frame}
