\begin{frame}{Learning Outcomes}
  After this session, you will be able to:
  \begin{itemize}
    \item[\textcolor{red}{$\blacktriangleright$}] Design effective prompts for various tasks
    \item[\textcolor{red}{$\blacktriangleright$}] Understand and apply few-shot and zero-shot prompting
    \item[\textcolor{red}{$\blacktriangleright$}] Implement chain-of-thought prompting
    \item[\textcolor{red}{$\blacktriangleright$}] Understand adversarial threats to LLMs
    \item[\textcolor{red}{$\blacktriangleright$}] Build RAG-based systems for enhanced LLM capabilities
  \end{itemize}
\end{frame}
